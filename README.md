# Narrative Radar - Centro de Mando Narrativo España

Proyecto de análisis de noticias y narrativas mediáticas en España.  
Incluye dashboard en Streamlit con tabs para:

- Radar Narrativo
- Radar Emocional
- Polarización
- Red de Actores
- Propagación
- Tendencias
- Cobertura del Gobierno
- Análisis Masivo de Medios
